📌 Overview

**SocialPulse Pro** is a fully responsive and modern **social media analytics dashboard** built using pure **HTML, CSS, and JavaScript**.

It simulates real-world social media management with:
- Analytics tracking 📊  
- Engagement monitoring ❤️  
- Content planning 📅  
- Audience insights 👥  
- AI content ideas 💡  
- Live Random User Generator 👤  

---

 ⚡ Key Functionalities (Highlighted Features)

 📊 1. Smart Analytics Dashboard
- Total Followers tracking
- Engagement Rate visualization
- Reach & CTR metrics
- Growth prediction charts (30-day forecast)
- Competitor performance comparison

---

 📸 2. Post Management System
- View all social posts in a clean UI
- Filter posts by platform:
  - Instagram 📷
  - Twitter 🐦
  - TikTok 🎵
- Search posts instantly
- Engagement stats (likes, comments, time)

---

 👥 3. Audience Insights Engine
- Age distribution analysis
- Gender breakdown charts
- Interactive data visualization using Chart.js

---

 ❤️ 4. Engagement Tracking System
- Real-time engagement trend graph
- Weekly performance monitoring
- Growth comparison metrics

---

 📅 5. Content Scheduling Planner
- Schedule posts for future publishing
- Visual timeline of planned content
- Quick schedule action system

---

 💬 6. Messaging Interface (UI Simulation)
- Direct message inbox layout
- User avatars & previews
- Reply interface (UI-based)

---

 💡 7. AI Content Idea Generator (Mock Engine)
- Random content idea suggestions
- One-click copy feature
- Helps in content strategy planning

---

 🔔 8. Notification System
- Live-style notification dropdown
- Unread notification counter
- Mark all as read feature

---

👤 9. Random User Generator (API Feature)
- Fetch real users from **RandomUser API**
- Displays:
  - Full name
  - Email address
  - Profile image
- Add user as mock follower
- Loading animation while fetching

---

 🌙 10. Dark Mode System
- Light/Dark theme toggle
- Saves user preference in localStorage
- Smooth UI transition

---

 📁 11. CSV Export Feature
- Export analytics data as CSV file
- Useful for reporting and data analysis
